## Navigation Architecture

1. **Tabs Layout** (InzintAI)
   - Multiple tabs with content
   - Featured projects section
   - Side CTA with image

2. **Columns Layout** (About, Services, Resources)
   - Multiple columns with categorized items
   - Nested submenus (Services only)
   - Bottom CTA

3. **Grid Layout** (Industries, Portfolio)
   - Card-based grid items
   - Icons and descriptions
